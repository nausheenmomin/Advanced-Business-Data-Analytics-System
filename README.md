#  Advanced Business Analytics Using the Superstore Dataset

##  Project Overview

This project demonstrates an end-to-end **Business Analytics workflow** using the **Superstore CSV dataset**. The project combines **Python (Google Colab)** for data preprocessing and exploratory data analysis (EDA) with **Power BI** for creating an interactive business dashboard.
The primary objective is to transform raw retail sales data into meaningful business insights that support informed decision-making.

---

##  Objectives

- Import and understand the Superstore dataset.
- Clean and preprocess the data.
- Perform Exploratory Data Analysis (EDA).
- Generate business insights using Python.
- Create interactive dashboards using Power BI.
- Analyze sales, profit, customers, regions, and product categories.
